// Projets

01
Stage Orange - 2025

Analyse de Sécurité Post-Quantique

Contexte

Dans le cadre de la recherche sur la transition vers la cryptographie post-quantique chez Orange, j'ai travaillé au sein de l'équipe R&D sur un projet visant à anticiper les menaces que représentent les futurs ordinateurs quantiques pour les systèmes cryptographiques actuels.

Objectifs

  • Développer une interface web d'analyse proactive des projets Git
  • Intégrer la détection automatisée des vulnérabilités logicielles et cryptographiques
  • Générer des rapports de sécurité détaillés avec recommandations de remédiation
  • Identifier les algorithmes cryptographiques vulnérables aux attaques quantiques

Processus et Methodologie

Développement en Python avec analyse statique de code, parsing AST pour détecter les appels cryptographiques, intégration d'APIs de détection de vulnérabilités (CVE), création d'une interface web intuitive pour visualiser les résultats et prioriser les corrections.

Resultats

  • Outil fonctionnel capable d'analyser des repositories Git en temps réel
  • Détection de plus de 50 patterns cryptographiques vulnérables
  • Rapports automatisés avec score de risque et recommandations priorisées
  • Documentation complète pour la continuité du projet
Python Git Analysis Post-Quantum Crypto Security API REST
02
Projet Personnel

Messagerie avec Cryptographie Quantique

Contexte

Projet académique réalisé dans le cadre de mes études à l'ESIEA, visant à explorer les algorithmes de cryptographie résistants aux ordinateurs quantiques et à les implémenter dans un cas d'usage concret : une messagerie sécurisée.

Objectifs

  • Implementer des algorithmes post-quantiques (Kyber, Sphinx+)
  • Créer un système de messagerie chiffré de bout en bout
  • Assurer l'échange sécurisé de clés résistant aux attaques quantiques
  • Documenter et comparer les performances avec les algorithmes classiques

Processus et Methodologie

Recherche bibliographique sur les standards NIST post-quantiques, implémentation en Python des protocoles d'échange de clés, développement d'une interface CLI pour les échanges de messages, tests de performance et benchmarking.

Resultats

  • Messagerie fonctionnelle avec chiffrement post-quantique
  • Rapport technique détaillé sur les algorithmes utilisés
Python Kyber Sphinx+ Cryptography CLI
03
Projet Personnel

IA avec Commande Vocale

Contexte

Projet personnel realise en parallele de mes etudes, visant a explorer les technologies de reconnaissance vocale et de synthese vocale pour creer un assistant intelligent.

Objectifs

  • Développer un assistant vocal intelligent en Python
  • Implémenter la reconnaissance vocale et la synthèse vocale
  • Créer un système de commandes naturelles interprétables
  • Intégrer des fonctionnalités domotiques et utilitaires

Processus et Methodologie

Utilisation de bibliothèques Python (SpeechRecognition, pyttsx3), conception d'une architecture modulaire pour les commandes, intégration d'APIs externes (météo, actualités), tests utilisateurs et itérations pour améliorer la reconnaissance.

Resultats

  • Assistant vocal fonctionnel avec plus de 20 commandes
  • Taux de reconnaissance vocale de 85% en environnement calme
  • Integration d'APIs externes (meteo, actualites, domotique)
  • Architecture modulaire permettant d'ajouter facilement de nouvelles commandes
Python Speech Recognition NLP API REST
